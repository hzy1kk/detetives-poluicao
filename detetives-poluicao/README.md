# Detetives da Poluição

Jogo educativo de Química ambiental — **2º EM Técnico, Turma TT** — Colégio Paulo de Tarso.

## Funcionalidades

- 4 casos investigativos com sorteio
- Dificuldade fácil / médio / difícil
- Pistas com desbloqueio e mini-perguntas
- Laboratório virtual
- Sistema de dicas e pontuação
- Modo treino
- Login da turma (senha: `detetive`)
- Modo professor (PIN: `detetive`) com CSV e importação
- PWA offline
- Relatórios em localStorage + export JSON/código

## Desenvolvimento

```bash
cd detetives-poluicao
npm install
npm run dev
```

## Deploy Vercel

1. Suba o repositório no GitHub
2. Importe no [vercel.com](https://vercel.com)
3. **Root Directory:** `detetives-poluicao`
4. Deploy automático
