Arquitetura

TecnologiaUso

Estrutura de pastas

detetives-poluicao/
├── src/
│   ├── components/   # Telas (login, jogo, professor…)
│   ├── data/         # Casos e config da escola
│   └── lib/          # Motor do jogo, áudio, storage
├── public/           # Logo, manual HTML
├── docs/             # Documentação PDF/HTML
└── vercel.json

Deploy (Vercel)

  1. Repositório: github.com/hzy1kk/detetives-poluicao
  2. Root: repositório com vercel.json na raiz
  3. Build: detetives-poluicao/dist
  4. Push em main → redeploy automático

Comandos

npm install
npm run dev      # desenvolvimento
npm run build    # produção
npm run docs:pdf # gerar PDFs da pasta docs/
